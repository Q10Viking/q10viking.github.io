## 模板语法

::: tip  

vue中页面访问数据

:::

```js
data{
    return {
        msg: "hello World",
        customTitle: 'title'
     
    }
}
```



### 属性

```html
<div :title="msg"> </div>
<!-- 动态属性 -->
<div :[customTitle]="msg">
```



### 页面

```html
<div>{{msg}}<div>
<!-- 只渲染一次，当msg变化的时候不在渲染 -->
<div v-once>{{msg}}<div>
```



### JS表达式

```vue
{{1+1}}
```
