## class样式

```js
data(){
	return {
		classString: 'red',		// 字符串
		classObject: {			// 对象
			red: false,
			green: false
		},
        classArray: ['red','green',{brown: true}] // 数组
	}
}
```



## 行内样式

```js
styleString: 'color: yellow',
styleOject: {
	color: 'orange'
}
```
