## 侦听器

price: <span>{{price}}</span> <button @click="price++"> 👉改变price的值 </button>

```js
data(){
    return {
        price: 0
    }
},
watch: {
    // 侦听price，当它发生变化时
    price(newValue,preValue){
        // ...
    }
}
```

::: tip

当computed和watcher都能实现的功能，建议使用computed，因为更加简洁

:::



----------

<script>
import {ref} from 'vue';

export default {
    setup(){
        return {
            price: ref(0)
        }
    },
    watch: {
        price(current,old){
            alert(`price原来是${old}现在变成${current}`);
        }
    }
}
</script>
