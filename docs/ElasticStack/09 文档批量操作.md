## 批量获取文档数据

批量获取文档数据是通过**_mget**的API来实现的

```
请求方式：GET
请求地址：_mget
功能说明 ： 可以通过ID批量获取不同index和type的数据
请求参数：
	docs : 文档数组参数
		_index : 指定index
		_type : 指定type
		_id : 指定id
		_source : 指定要查询的字段
```



### 在URL中不指定index

```json
GET _mget
{
  "docs": [
    {"_index": "es_db","_type": "_doc","_id": 1},
    {"_index": "es_db","_type": "_doc","_id": 2}
  ]
}
```

![image-20220812220614521](/images/elasticsearch/image-20220812220614521.png)

### 在URL中指定index

```json
GET /es_db/_mget
{
  "docs": [
    {"_type": "_doc","_id": 3},
    {"_type": "_doc","_id": 4}
  ]
}
```
