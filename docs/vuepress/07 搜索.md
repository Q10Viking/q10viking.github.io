## docsearch

将 [Algolia DocSearch在新窗口打开](https://docsearch.algolia.com/) 集成到 VuePress 中，为你的文档网站提供搜索功能。

```sh
npm i -D @vuepress/plugin-docsearch@next
```

将我们的博客网站加入 [DocSearch项目](https://docsearch.algolia.com/apply/) ，随后会发送相关信息给我们

配置

```js
  plugins:[
    [ '@vuepress/docsearch',{
      apiKey: '',
      indexName: ''
    }]
  ]
```
