## 博客配置

```sh
# 安装Google分析插件
npm i -D @vuepress/plugin-google-analytics@next
```

在config.js配置

```js
module.exports = {
  plugins: [
    [
      '@vuepress/plugin-google-analytics',
      {
        id: 'G-XXXXXXXXXX',
      },
    ],
  ],
}
```
