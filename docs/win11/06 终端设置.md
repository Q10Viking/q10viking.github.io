## 拆分窗格操作

[Windows 终端窗格 | Microsoft Learn](https://learn.microsoft.com/zh-cn/windows/terminal/panes)

| 描述         | 操作                             |
| ------------ | -------------------------------- |
| 关闭窗格     | ctrl+shift+w                     |
| 切换窗格     | alt+上下左右箭头                 |
| 调整窗格大小 | alt+shift+上下左右箭头           |
| 拆分当前窗格 | 点击当前选项卡然后选择拆分选项卡 |
| 拆分新增窗格 | 按住alt+点击要新增的选显卡       |
