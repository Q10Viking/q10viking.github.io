## Go tools

```sh
# 在当前目录下执行,会执行所有的*_test.go文件
go test  
# go run main.go 执行文件
go run
go build
# 列出所有下载的依赖
go list
# 环境变量
go env

#初始化一个项目
go mod init github.com/q10viking/coursera
```
