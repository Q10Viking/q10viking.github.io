## MyBatis介绍

应用数据是在内存中的，当断电之后，为了保证数据不丢失，需要将数据进行持久化。而MyBatis是一款很优秀的持久层框架。

MyBatis免除了几乎所有的JDBC代码以及设置参数和获取结果集的工作。
