## kubectl命令使用

[命令行工具 (kubectl) | Kubernetes](https://kubernetes.io/zh-cn/docs/reference/kubectl/)

kubectl是apiserver的客户端工具，工作在命令行下，能够连接apiserver实现各种增删改查等操作

```sh
# 帮助命令 K8S的各种命令帮助文档做得非常不错，遇到问题可以多查help帮助
kubectl -h
```
