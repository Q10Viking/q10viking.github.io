# 实战安装Ubuntu

在win11中集成了wsl命令，可以不用借助VMware,来直接安装Linux操作系统

PS C:\Users\11930\Desktop> wsl --install
适用于 Linux 的 Windows 子系统已安装。

以下是可安装的有效分发的列表。
请使用“wsl --install -d <分发>”安装。

# wsl关闭发行版⭐

  NAME      STATE           VERSION
* Ubuntu    Running         2
PS C:\> wsl -t Ubuntu # 关闭
PS C:\> wsl -l -v
  NAME      STATE           VERSION
* Ubuntu    Stopped         2
