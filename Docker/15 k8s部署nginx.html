搭建好了k8s，我们来部署一个nginx服务，来初步使用以下k8s

# 部署一个nginx

在k8s-master机器上执行

# 创建一次deployment部署
kubectl create deployment nginx --image=nginx
# --port是service的虚拟ip对应的端口，系统会随机暴露一个端口
kubectl expose deployment nginx --port=80 --type=NodePort

# 访问

虽然nginx的pod部署在了k8s-node1节点对应的ip地址为http://192.168.135.132:31448/open in new window，但是,我们能够从master，k8s-node2上都能访问到。因为kubunetes默认安装了nginx，它有反向代理的功能。

访问Nginx地址 http://任意节点的ip:图中Nginx的对外映射端口

http://192.168.135.130:31448/open in new window

http://192.168.135.132:31448/open in new window

http://192.168.135.135:31448/open in new window

# windows修改hosts文件

上面的过程中使用ip来进行访问非常不方便，在windows下修改hosts文件。这里用的软件是SwitchHosts

修改之后就能通过域名进行访问了

http://k8s-master:31448/open in new window

http://k8s-node1:31448/open in new window

http://k8s-node2:31448/open in new window
