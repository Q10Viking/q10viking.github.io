# key的作用⭐

TIP

1️⃣ 循环中key的作用：辅助Vue,让其渲染时性能更高

2️⃣ key是 Vue 识别节点的一个通用机制，key 并不仅与 v-for 特别关联

List Rendering maintaining-state | Vue.js (vuejs.org)open in new window

没有key的情况，也就没有识别节点，先勾选了Vue,然后点击添加，发现Vue不再被勾选，反而变成了Java成勾选状态

See the Pen v-for-example-noKey by 静默 (@Q10Viking) on CodePen.

有key的情况，识别节点，先勾选了Vue,然后点击添加，发现Vue任然被勾选，注意绑定的key为item,而不是value

查看代码
<div id="root">
  <input v-model="name" />
  <button @click="handleClick">添加</button>
  <ul>
    <!-- 如果key任然绑定的时index,还是会出现上面的情况  -->
    <!-- <li v-for="(item,index) in listArray" :key="index">  -->
    <li v-for="(item,index) in listArray" :key="item">
      <input type="checkbox"/> {{item.name}}
    </li>
  </ul>
</div>
