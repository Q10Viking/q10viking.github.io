# max-height&overflow-y理解

TIP

用户列表设置高度采用max-height.当用户数量较少得时候,内容完全由内容撑开，当用户数量很多，超出容器得时候，max-height就会生效。这是overflow-y:auto；就会出现滚动条

.userlist{
      max-height: 400px;
      overflow-y: auto;
}
