# е…Ёеӯ—ж®өжҺ’еәҸ

Extra иҝҷдёӘеӯ—ж®өдёӯзҡ„вҖңUsing filesortвҖқиЎЁзӨәзҡ„е°ұжҳҜйңҖиҰҒжҺ’еәҸпјҢMySQL дјҡз»ҷжҜҸдёӘзәҝзЁӢеҲҶй…ҚдёҖеқ—еҶ…еӯҳз”ЁдәҺжҺ’еәҸпјҢз§°дёә sort_buffer

иҝҷдёӘиҜӯеҸҘжү§иЎҢжөҒзЁӢеҰӮдёӢжүҖзӨә пјҡ

  1. еҲқе§ӢеҢ– sort_bufferпјҢзЎ®е®ҡж”ҫе…Ҙ nameгҖҒcityгҖҒage иҝҷдёүдёӘеӯ—ж®өпјӣ
  2. д»Һзҙўеј• city жүҫеҲ°з¬¬дёҖдёӘж»Ўи¶і city='жқӯе·һвҖҷжқЎд»¶зҡ„дё»й”® idпјҢд№ҹе°ұжҳҜеӣҫдёӯзҡ„ ID_Xпјӣ
  3. еҲ°дё»й”® id зҙўеј•еҸ–еҮәж•ҙиЎҢпјҢеҸ– nameгҖҒcityгҖҒage дёүдёӘеӯ—ж®өзҡ„еҖјпјҢеӯҳе…Ҙ sort_buffer дёӯпјӣ
  4. д»Һзҙўеј• city еҸ–дёӢдёҖдёӘи®°еҪ•зҡ„дё»й”® idпјӣ
  5. йҮҚеӨҚжӯҘйӘӨ 3гҖҒ4 зӣҙеҲ° city зҡ„еҖјдёҚж»Ўи¶іжҹҘиҜўжқЎд»¶дёәжӯўпјҢеҜ№еә”зҡ„дё»й”® id д№ҹе°ұжҳҜеӣҫдёӯзҡ„ ID_Yпјӣ
  6. еҜ№ sort_buffer дёӯзҡ„ж•°жҚ®жҢүз…§еӯ—ж®ө name еҒҡеҝ«йҖҹжҺ’еәҸпјӣ
  7. жҢүз…§жҺ’еәҸз»“жһңеҸ–еүҚ 1000 иЎҢиҝ”еӣһз»ҷе®ўжҲ·з«ҜгҖӮ

жҲ‘д»¬жҡӮдё”жҠҠиҝҷдёӘжҺ’еәҸиҝҮзЁӢпјҢз§°дёәе…Ёеӯ—ж®өжҺ’еәҸ

вҖңжҢү name жҺ’еәҸвҖқиҝҷдёӘеҠЁдҪңпјҢеҸҜиғҪеңЁеҶ…еӯҳдёӯе®ҢжҲҗпјҢд№ҹеҸҜиғҪйңҖиҰҒдҪҝз”ЁеӨ–йғЁжҺ’еәҸ(дёҙж—¶ж–Үд»¶)пјҢиҝҷеҸ–еҶідәҺжҺ’еәҸжүҖйңҖзҡ„еҶ…еӯҳе’ҢеҸӮж•° sort_buffer_size

# rowidжҺ’еәҸ

еңЁдёҠйқўиҝҷдёӘз®—жі•иҝҮзЁӢйҮҢйқўпјҢеҸӘеҜ№еҺҹиЎЁзҡ„ж•°жҚ®иҜ»дәҶдёҖйҒҚпјҢеү©дёӢзҡ„ж“ҚдҪңйғҪжҳҜеңЁ sort_buffer е’Ңдёҙж—¶ж–Үд»¶дёӯжү§иЎҢзҡ„гҖӮдҪҶиҝҷдёӘз®—жі•жңүдёҖдёӘй—®йўҳпјҢе°ұжҳҜеҰӮжһңжҹҘиҜўиҰҒиҝ”еӣһзҡ„еӯ—ж®өеҫҲеӨҡзҡ„иҜқпјҢйӮЈд№Ҳ sort_buffer йҮҢйқўиҰҒж”ҫзҡ„еӯ—ж®өж•°еӨӘеӨҡпјҢиҝҷж ·еҶ…еӯҳйҮҢиғҪеӨҹеҗҢж—¶ж”ҫдёӢзҡ„иЎҢж•°еҫҲе°‘пјҢиҰҒеҲҶжҲҗеҫҲеӨҡдёӘдёҙж—¶ж–Үд»¶пјҢжҺ’еәҸзҡ„жҖ§иғҪдјҡеҫҲе·®гҖӮ

жүҖд»ҘеҰӮжһңеҚ•иЎҢеҫҲеӨ§пјҢиҝҷдёӘж–№жі•ж•ҲзҺҮдёҚеӨҹеҘҪ

max_length_for_sort_dataпјҢжҳҜ MySQL дёӯдё“й—ЁжҺ§еҲ¶з”ЁдәҺжҺ’еәҸзҡ„иЎҢж•°жҚ®зҡ„й•ҝеәҰзҡ„дёҖдёӘеҸӮж•°гҖӮе®ғзҡ„ж„ҸжҖқжҳҜпјҢеҰӮжһңеҚ•иЎҢзҡ„й•ҝеәҰи¶…иҝҮиҝҷдёӘеҖјпјҢMySQL е°ұи®ӨдёәеҚ•иЎҢеӨӘеӨ§пјҢиҰҒжҚўдёҖдёӘз®—жі•гҖӮ

cityгҖҒnameгҖҒage иҝҷдёүдёӘеӯ—ж®өзҡ„е®ҡд№үжҖ»й•ҝеәҰжҳҜ 36пјҢжҲ‘жҠҠ max_length_for_sort_data и®ҫзҪ®дёә 16пјҢжҲ‘д»¬еҶҚжқҘзңӢзңӢи®Ўз®—иҝҮзЁӢжңүд»Җд№Ҳж”№еҸҳгҖӮ

ж–°зҡ„з®—жі•ж”ҫе…Ҙ sort_buffer зҡ„еӯ—ж®өпјҢеҸӘжңүиҰҒжҺ’еәҸзҡ„еҲ—пјҲеҚі name еӯ—ж®өпјүе’Ңдё»й”® idгҖӮ

жҺ’еәҸзҡ„з»“жһңе°ұеӣ дёәе°‘дәҶ city е’Ң age еӯ—ж®өзҡ„еҖјпјҢдёҚиғҪзӣҙжҺҘиҝ”еӣһдәҶпјҢж•ҙдёӘжү§иЎҢжөҒзЁӢе°ұеҸҳжҲҗеҰӮдёӢжүҖзӨәзҡ„ж ·еӯҗпјҡ

  1. еҲқе§ӢеҢ– sort_bufferпјҢзЎ®е®ҡж”ҫе…ҘдёӨдёӘеӯ—ж®өпјҢеҚі name е’Ң idпјӣ
  2. д»Һзҙўеј• city жүҫеҲ°з¬¬дёҖдёӘж»Ўи¶і city='жқӯе·һвҖҷжқЎд»¶зҡ„дё»й”® idпјҢд№ҹе°ұжҳҜеӣҫдёӯзҡ„ ID_Xпјӣ
  3. еҲ°дё»й”® id зҙўеј•еҸ–еҮәж•ҙиЎҢпјҢеҸ– nameгҖҒid иҝҷдёӨдёӘеӯ—ж®өпјҢеӯҳе…Ҙ sort_buffer дёӯпјӣ
  4. д»Һзҙўеј• city еҸ–дёӢдёҖдёӘи®°еҪ•зҡ„дё»й”® idпјӣ
  5. йҮҚеӨҚжӯҘйӘӨ 3гҖҒ4 зӣҙеҲ°дёҚж»Ўи¶і city='жқӯе·һвҖҷжқЎд»¶дёәжӯўпјҢд№ҹе°ұжҳҜеӣҫдёӯзҡ„ ID_Yпјӣ
  6. еҜ№ sort_buffer дёӯзҡ„ж•°жҚ®жҢүз…§еӯ—ж®ө name иҝӣиЎҢжҺ’еәҸпјӣ
  7. йҒҚеҺҶжҺ’еәҸз»“жһңпјҢеҸ–еүҚ 1000 иЎҢпјҢе№¶жҢүз…§ id зҡ„еҖјеӣһеҲ°еҺҹиЎЁдёӯеҸ–еҮә cityгҖҒname е’Ң age дёүдёӘеӯ—ж®өиҝ”еӣһз»ҷе®ўжҲ·з«ҜгҖӮ

иҝҷдёӘжү§иЎҢжөҒзЁӢзҡ„зӨәж„ҸеӣҫеҰӮдёӢпјҢжҲ‘жҠҠе®ғз§°дёә rowid жҺ’еәҸгҖӮ

# дҪҝз”ЁиҒ”еҗҲзҙўеј•иҝӣиЎҢдјҳеҢ–

еҰӮжһңиғҪеӨҹдҝқиҜҒд»Һ city иҝҷдёӘзҙўеј•дёҠеҸ–еҮәжқҘзҡ„иЎҢпјҢеӨ©з„¶е°ұжҳҜжҢүз…§ name йҖ’еўһжҺ’еәҸзҡ„иҜқпјҢжҳҜдёҚжҳҜе°ұеҸҜд»ҘдёҚз”ЁеҶҚжҺ’еәҸдәҶе‘ў

alter table t add index city_user(city, name);
1

еңЁиҝҷдёӘзҙўеј•йҮҢйқўпјҢжҲ‘д»¬дҫқз„¶еҸҜд»Ҙз”Ёж ‘жҗңзҙўзҡ„ж–№ејҸе®ҡдҪҚеҲ°з¬¬дёҖдёӘж»Ўи¶і city='жқӯе·һвҖҷзҡ„и®°еҪ•пјҢе№¶дё”йўқеӨ–зЎ®дҝқдәҶпјҢжҺҘдёӢжқҘжҢүйЎәеәҸеҸ–вҖңдёӢдёҖжқЎи®°еҪ•вҖқзҡ„йҒҚеҺҶиҝҮзЁӢдёӯпјҢеҸӘиҰҒ city зҡ„еҖјжҳҜжқӯе·һпјҢname зҡ„еҖје°ұдёҖе®ҡжҳҜжңүеәҸзҡ„гҖӮ

иҝҷж ·ж•ҙдёӘжҹҘиҜўиҝҮзЁӢзҡ„жөҒзЁӢе°ұеҸҳжҲҗдәҶпјҡ

  1. д»Һзҙўеј• (city,name) жүҫеҲ°з¬¬дёҖдёӘж»Ўи¶і city='жқӯе·һвҖҷжқЎд»¶зҡ„дё»й”® idпјӣ
  2. еҲ°дё»й”® id зҙўеј•еҸ–еҮәж•ҙиЎҢпјҢеҸ– nameгҖҒcityгҖҒage дёүдёӘеӯ—ж®өзҡ„еҖјпјҢдҪңдёәз»“жһңйӣҶзҡ„дёҖйғЁеҲҶзӣҙжҺҘиҝ”еӣһпјӣ
  3. д»Һзҙўеј• (city,name) еҸ–дёӢдёҖдёӘи®°еҪ•дё»й”® idпјӣ
  4. йҮҚеӨҚжӯҘйӘӨ 2гҖҒ3пјҢзӣҙеҲ°жҹҘеҲ°з¬¬ 1000 жқЎи®°еҪ•пјҢжҲ–иҖ…жҳҜдёҚж»Ўи¶і city='жқӯе·һвҖҷжқЎд»¶ж—¶еҫӘзҺҜз»“жқҹ

еҸҜд»ҘзңӢеҲ°пјҢиҝҷдёӘжҹҘиҜўиҝҮзЁӢдёҚйңҖиҰҒдёҙж—¶иЎЁпјҢд№ҹдёҚйңҖиҰҒжҺ’еәҸгҖӮжҺҘдёӢжқҘпјҢжҲ‘д»¬з”Ё explain зҡ„з»“жһңжқҘеҚ°иҜҒдёҖдёӢгҖӮ

Extra еӯ—ж®өдёӯжІЎжңү Using filesort дәҶпјҢд№ҹе°ұжҳҜдёҚйңҖиҰҒжҺ’еәҸдәҶгҖӮеӣ дёә (city,name) иҝҷдёӘиҒ”еҗҲзҙўеј•жң¬иә«жңүеәҸ

# дҪҝз”ЁиҰҶзӣ–зҙўеј•дјҳеҢ–

иҰҶзӣ–зҙўеј•жҳҜжҢҮпјҢзҙўеј•дёҠзҡ„дҝЎжҒҜи¶іеӨҹж»Ўи¶іжҹҘиҜўиҜ·жұӮпјҢдёҚйңҖиҰҒеҶҚеӣһеҲ°дё»й”®зҙўеј•дёҠеҺ»еҸ–ж•°жҚ®гҖӮ

жҢүз…§иҰҶзӣ–зҙўеј•зҡ„жҰӮеҝөпјҢжҲ‘д»¬еҸҜд»ҘеҶҚдјҳеҢ–дёҖдёӢиҝҷдёӘжҹҘиҜўиҜӯеҸҘзҡ„жү§иЎҢжөҒзЁӢгҖӮ

й’ҲеҜ№иҝҷдёӘжҹҘиҜўпјҢжҲ‘д»¬еҸҜд»ҘеҲӣе»әдёҖдёӘ cityгҖҒname е’Ң age зҡ„иҒ”еҗҲзҙўеј•пјҢеҜ№еә”зҡ„ SQL иҜӯеҸҘе°ұжҳҜ

  1. д»Һзҙўеј• (city,name,age) жүҫеҲ°з¬¬дёҖдёӘж»Ўи¶і city='жқӯе·һвҖҷжқЎд»¶зҡ„и®°еҪ•пјҢеҸ–еҮәе…¶дёӯзҡ„ cityгҖҒname е’Ң age иҝҷдёүдёӘеӯ—ж®өзҡ„еҖјпјҢдҪңдёәз»“жһңйӣҶзҡ„дёҖйғЁеҲҶзӣҙжҺҘиҝ”еӣһпјӣ
  2. д»Һзҙўеј• (city,name,age) еҸ–дёӢдёҖдёӘи®°еҪ•пјҢеҗҢж ·еҸ–еҮәиҝҷдёүдёӘеӯ—ж®өзҡ„еҖјпјҢдҪңдёәз»“жһңйӣҶзҡ„дёҖйғЁеҲҶзӣҙжҺҘиҝ”еӣһпјӣ
  3. йҮҚеӨҚжү§иЎҢжӯҘйӘӨ 2пјҢзӣҙеҲ°жҹҘеҲ°з¬¬ 1000 жқЎи®°еҪ•пјҢжҲ–иҖ…жҳҜдёҚж»Ўи¶і city='жқӯе·һвҖҷжқЎд»¶ж—¶еҫӘзҺҜз»“жқҹ

з„¶еҗҺпјҢжҲ‘д»¬еҶҚжқҘзңӢзңӢ explain зҡ„з»“жһңгҖӮ

еҸҜд»ҘзңӢеҲ°пјҢExtra еӯ—ж®өйҮҢйқўеӨҡдәҶвҖңUsing indexвҖқпјҢиЎЁзӨәзҡ„е°ұжҳҜдҪҝз”ЁдәҶиҰҶзӣ–зҙўеј•пјҢжҖ§иғҪдёҠдјҡеҝ«еҫҲеӨҡгҖӮ

еҪ“з„¶пјҢиҝҷйҮҢе№¶дёҚжҳҜиҜҙиҰҒдёәдәҶжҜҸдёӘжҹҘиҜўиғҪз”ЁдёҠиҰҶзӣ–зҙўеј•пјҢе°ұиҰҒжҠҠиҜӯеҸҘдёӯж¶үеҸҠзҡ„еӯ—ж®өйғҪе»әдёҠиҒ”еҗҲзҙўеј•пјҢжҜ•з«ҹзҙўеј•иҝҳжҳҜжңүз»ҙжҠӨд»Јд»·зҡ„гҖӮиҝҷжҳҜдёҖдёӘйңҖиҰҒжқғиЎЎзҡ„еҶіе®ҡ

# filesortж–Үд»¶жҺ’еәҸж–№ејҸ

  • еҚ•и·ҜжҺ’еәҸпјҡжҳҜдёҖж¬ЎжҖ§еҸ–еҮәж»Ўи¶іжқЎд»¶иЎҢзҡ„жүҖжңүеӯ—ж®өпјҢз„¶еҗҺеңЁsort bufferдёӯиҝӣиЎҢжҺ’еәҸпјӣз”Ёtraceе·Ҙе…·еҸҜд»ҘзңӢеҲ°sort_modeдҝЎжҒҜйҮҢжҳҫзӨә< sort_key, additional_fields >жҲ–иҖ…< sort_key, packed_additional_fields >
  • еҸҢи·ҜжҺ’еәҸпјҲеҸҲеҸ«еӣһиЎЁжҺ’еәҸжЁЎејҸпјүпјҡжҳҜйҰ–е…Ҳж №жҚ®зӣёеә”зҡ„жқЎд»¶еҸ–еҮәзӣёеә”зҡ„жҺ’еәҸеӯ—ж®өе’ҢеҸҜд»ҘзӣҙжҺҘе®ҡдҪҚиЎҢж•°жҚ®зҡ„иЎҢ IDпјҢз„¶еҗҺеңЁ sort buffer дёӯиҝӣиЎҢжҺ’еәҸпјҢжҺ’еәҸе®ҢеҗҺйңҖиҰҒеҶҚж¬ЎеҸ–еӣһе…¶е®ғйңҖиҰҒзҡ„еӯ—ж®өпјӣз”Ёtraceе·Ҙе…·еҸҜд»ҘзңӢеҲ°sort_modeдҝЎжҒҜйҮҢжҳҫзӨә< sort_key, rowid >

MySQL йҖҡиҝҮжҜ”иҫғзі»з»ҹеҸҳйҮҸ max_length_for_sort_data(й»ҳи®Ө1024еӯ—иҠӮ) зҡ„еӨ§е°Ҹе’ҢйңҖиҰҒжҹҘиҜўзҡ„еӯ—ж®өжҖ»еӨ§е°ҸжқҘеҲӨж–ӯдҪҝз”Ёе“Әз§ҚжҺ’еәҸжЁЎејҸгҖӮ

  • еҰӮжһң еӯ—ж®өзҡ„жҖ»й•ҝеәҰе°ҸдәҺmax_length_for_sort_data пјҢйӮЈд№ҲдҪҝз”Ё еҚ•и·ҜжҺ’еәҸжЁЎејҸпјӣ
  • еҰӮжһң еӯ—ж®өзҡ„жҖ»й•ҝеәҰеӨ§дәҺmax_length_for_sort_data пјҢйӮЈд№ҲдҪҝз”Ё еҸҢи·ҜжҺ’еәҸжЁЎВ·ејҸ
