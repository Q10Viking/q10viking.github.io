# Inheritance继承性

TIP

css中有一部分属性可以被继承,盒子属性，如宽和高不能被继承

如：文字和文本相关的属性，可以被继承

玩法：可以直接在body中设置字体，来控制整个页面的字体显示

body {
	font-family: sans-serif;
}

可以作为属性填入

	font-family: inherid;

# 层叠性

TIP

样式直接作用在标签上

# 优先级

权重比较

  1. 比较id选择器，类选择，标签选择器的个数
  2. 权重相同，则选择后编写的代码样式

# !important提升权重

TIP

Overwrites specifity and all other selectors

在直接作用在标签上的属性，通过!important可以提高权重

div {
	color: red !important;
}
