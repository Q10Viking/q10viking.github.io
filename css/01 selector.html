# 1.3 清除默认样式（重要）

  1. 清除样式能够在开发中避免浏览器突然添加的默认样式（这会影响开发的预期效果）

从效率上来看并集选择器在书写上繁琐，但是效率比较高

// 通配符*全选择器
*{
	margin: 0;
    padding: 0;
}

// 并集选择器
div,p{
   margin: 0;
   padding: 0; 
}

# 属性选择器

# 普通选择器

E[attr="val"]
1

# ^属性开头

E[attr^="val"]  选择拥有attr属性且属性值为val开头的E元素
1

# $属性结束

E[attr$="val"]  选择拥有attr属性且属性值以val结束的E元素
1

# *包含属性

E[attr*="val"]   选择拥有attr属性且属性值中包含val的E元素
1
