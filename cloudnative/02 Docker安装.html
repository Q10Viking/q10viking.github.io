以下操作在本地虚拟机centos7 192.168.187.132 操作

# 准备工作

TIP

1️⃣ 社区版（Community Edition，缩写为 CE）

2️⃣ 企业版（Enterprise Edition，缩写为 EE）

确保 yum 包更新到最新

yum -y update
1

安装相关工具

yum install -y yum-utils
1

设置yum源，并更新 yum 的包索引

# 官方的yum源
yum-config-manager --add-repo https://download.docker.com/linux/centos/docker-ce.repo
# 这里使用阿里的云源，来加快访问速度
yum-config-manager --add-repo http://mirrors.aliyun.com/docker-ce/linux/centos/docker-ce.repo
yum makecache fast

5

查看所有仓库中所有docker版本，并选择特定版本安装

yum list docker-ce --showduplicates | sort -r
1

# 安装docker

TIP

用指定版本的方式安装

yum install -y docker-ce-3:19.03.9-3.el7.x86_64
1

# 启动Docker

systemctl start docker
# 加入开机启动
systemctl enable docker

# 配置docke镜像加速器

TIP

借助阿里云的镜像加速器，登录阿里云(https://cr.console.aliyun.com/#/accelerator)

sudo mkdir -p /etc/docker
sudo tee /etc/docker/daemon.json <<-'EOF'
{
  "registry-mirrors": ["https://u8x09gus.mirror.aliyuncs.com"]
}
EOF
sudo systemctl daemon-reload
sudo systemctl restart docker

# 卸载docker

yum remove -y docker*
rm -rf /etc/systemd/system/docker.service.d
rm -rf /var/lib/docker
rm -rf /var/run/docker
